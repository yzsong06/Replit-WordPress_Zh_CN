## 使用说明
- replit环境选择`Bash`
- 不建议使用免费版部署，会泄漏数据库源文件

```bash
bash <(curl -s https://raw.githubusercontent.com/yzsong06/Replit-Wordpress_zh_CN/main/install.sh)
```
- 执行上面脚本
- 最后点击Run运行项目即可。
